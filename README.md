# Exposition

**Exposition** est  outil web gratuit destiné aux professionnels formés à la thérapie comportementale et cognitive et à la conduite des interventions par exposition.

Il permet de préparer, conduire, documenter et consulter des exercices d’exposition en séance.

## Fonctionnalités

L’outil comprend plusieurs modules :

- gestion de dossiers patients pseudonymisés ;
- création d’un OptEx ;
- création de hiérarchies d’exposition ;
- rédaction de scripts :
  - souvenir traumatique ;
  - scénario catastrophe ;
- préparation d’expositions fondées sur :
  - l’habituation ;
  - l’invalidation des attentes ;
- enregistrement de mesures pendant l’exposition :
  - intensité de l’émotion cible ;
  - probabilité d’occurrence de l’issue redoutée ;
  - observations cliniques horodatées ;
- graphiques d’évolution ;
- débriefing structuré ;
- historique des exercices ;
- génération de comptes rendus PDF ;
- exportation et restauration des données.

## Public concerné

Cet outil est destiné aux psychologues, psychothérapeutes et autres professionnels de santé formés à la thérapie par exposition.

Il ne constitue pas :

- un dispositif médical ;
- un outil diagnostique ;
- un programme thérapeutique autonome ;
- un substitut à la formation clinique, à la supervision ou au jugement du professionnel.

Il ne doit pas être utilisé de manière autonome par une personne ne maîtrisant pas les principes, indications, précautions et modalités de conduite de l’exposition.

## Confidentialité et stockage des données

Aucune donnée saisie dans l’application n’est transmise à GitHub ni à un serveur distant.

Les données sont enregistrées localement dans le navigateur de l’utilisateur, notamment à l’aide d’IndexedDB et de `localStorage`.

Conséquences importantes :

- les données ne sont pas automatiquement synchronisées entre les appareils ;
- les données enregistrées sur un ordinateur ne sont pas automatiquement disponibles sur un iPad ;
- la suppression des données du site ou du stockage du navigateur peut entraîner la perte de toutes les informations ;
- une navigation privée peut empêcher la conservation durable des données ;
- des sauvegardes régulières doivent être réalisées depuis l’onglet **Sauvegarde**.

Utilisez uniquement des codes pseudonymisés et ne saisissez pas directement l’identité complète des patients.
